AutoShine

Política de Privacidade

Esta política resume como o AutoShine usa dados necessários para cadastro, login, agendamento, avaliação e gestão das lojas.

Dados coletados

Podemos tratar nome, e-mail, CPF, telefone, login de parceiro, CNPJ, dados de agendamento, avaliações e registros técnicos de segurança.

Finalidades

Os dados são usados para autenticar usuários, validar cadastros, reservar horários, exibir avaliações, administrar lojas e proteger a plataforma.

Compartilhamento

Dados de agendamento podem ser disponibilizados ao parceiro responsável pela loja para execução do serviço. Validações oficiais podem consultar provedores externos configurados no servidor.

Segurança

Senhas são armazenadas com hash e rotas sensíveis possuem limite de tentativas.

Direitos do usuário

O usuário pode solicitar correção, exclusão ou informações sobre seus dados pelos canais oficiais do projeto.
